# Contributing

Thank you for contributing to the Unit Conversion API.

## Getting started

1. Install the [.NET 9 SDK](https://dotnet.microsoft.com/download).
2. Clone the repository.
3. Run `dotnet restore`, `dotnet build`, and `dotnet test`.

## Pull request expectations

- Keep changes focused and include tests for new behavior.
- Ensure `dotnet test` passes locally before opening a PR.
- Follow the formatting rules defined in [`.editorconfig`](.editorconfig).
- Update [`README.md`](README.md) when behavior, endpoints, or setup steps change.

## Code style

- Use 4-space indentation for C# files.
- Place opening braces on new lines.
- Prefer clear names and small, focused classes over large utility files.
